# Audit Evidence Checklist: IAM/RBAC Access Review Agent

## Checklist Purpose

This checklist identifies the evidence that would be needed to support an IAM/RBAC access review in an audit-ready manner.

This is a simulated portfolio artifact using fictional data only.

## Control Objective

User access should be reviewed periodically to confirm access is appropriate based on job role, department, employment status, and approved RBAC rules.

## Evidence Checklist

| Evidence ID | Evidence Item | Purpose | Owner | Status |
|---|---|---|---|---|
| E-001 | Approved RBAC matrix | Confirms approved role-based access rules | IAM / GRC | Sample Included |

## Evidence Quality Questions

Use these questions to evaluate whether evidence is complete and audit-ready:

- Does the evidence clearly show the review period?
- Does the evidence identify who performed the review?
- Does the evidence show which users were reviewed?
- Does the evidence show what access was reviewed?
- Does the evidence include the approved RBAC baseline?
- Does the evidence show exceptions and decisions?
- Does the evidence show manager or control owner approval?
- Does the evidence show remediation was completed when required?
- Does the evidence show the date remediation occurred?
- Does the evidence avoid exposing sensitive production data?

## Example Audit Request

**Request:** Provide evidence that user access was reviewed for Q3-2026 and that users with inappropriate access were identified and routed for remediation.

## Example Evidence Response

The following evidence would support the request:

1. Approved RBAC matrix used as the access baseline.
2. User access extract for Q3-2026.
3. Access review findings report.
4. Manager approval or rejection records.
5. Remediation tickets for removed access.
6. Final review sign-off from IAM or control owner.

## Human Approval Requirement

The AI agent may assist with evidence organization and findings summaries, but a human control owner must review and approve final evidence before it is submitted for audit.
