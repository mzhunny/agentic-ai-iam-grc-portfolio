# Portfolio Roadmap

## Portfolio Goal

Build a recruiter-ready Agentic AI portfolio focused on IAM, RBAC, GRC, audit evidence, and AI security governance.

This portfolio is designed to show practical understanding of identity access controls, risk-based testing, compliance documentation, and safe AI-assisted security workflows.

## Phase 1: Portfolio Foundation

Status: In Progress

Deliverables:

- Main README
- Portfolio roadmap
- Project folder structure
- Starter README files for each project
- Templates folder
- Glossary folder

## Phase 2: IAM/RBAC Access Review Agent

Primary goal: Build a simulated AI agent workflow that reviews user access and flags excessive permissions, missing access, and access governance risks.

Deliverables:

- Business requirements
- User stories
- RBAC matrix
- Sample user access data
- Agent workflow
- Test cases
- Risk findings report
- Audit evidence checklist

## Phase 3: SoD Conflict Detection Agent

Primary goal: Demonstrate how an AI-assisted workflow can identify conflicting permissions based on defined Segregation of Duties rules.

Deliverables:

- SoD rules matrix
- Sample access conflict data
- Agent workflow
- Test cases
- Remediation report

## Phase 4: GRC Policy Mapping Agent

Primary goal: Demonstrate AI-assisted control mapping from sample policies to common frameworks.

Deliverables:

- Sample access control policy
- NIST control mapping
- ISO control mapping
- Gap analysis report
- QA validation test cases

## Phase 5: Audit Evidence Review Agent

Primary goal: Demonstrate audit-readiness thinking by validating whether evidence meets control requirements.

Deliverables:

- Audit request examples
- Evidence checklist
- Control owner questions
- Validation test cases
- Audit readiness summary

## Phase 6: Agentic AI Security Controls

Primary goal: Demonstrate understanding of AI agent risks, human approval gates, logging, monitoring, tool permissions, and prompt injection concerns.

Deliverables:

- Agent risk assessment
- Threat model
- Human-in-the-loop control design
- Logging and monitoring requirements
- Security test scenarios

## Final Portfolio Outcome

By the end of this roadmap, the portfolio should demonstrate readiness for roles such as:

- IAM Business Analyst
- IAM Tester
- Security QA Analyst
- GRC Analyst
- Cybersecurity Business Analyst
- Agentic AI Security Analyst
