# SoD Conflict Detection Agent

## Project Summary

This project demonstrates a simulated Agentic AI workflow that identifies Segregation of Duties conflicts in user permissions.

## Business Problem

Users should not have combinations of access that allow them to perform conflicting or high-risk activities without oversight. For example, a user should not be able to both create a vendor and approve vendor payments.

## Agent Use Case

The AI agent reviews fictional permission sets and compares them against a defined SoD rules matrix. It flags risky permission combinations and recommends remediation steps.
