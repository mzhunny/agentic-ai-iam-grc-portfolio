# Audit Evidence Review Agent

## Project Summary

This project demonstrates a simulated Agentic AI workflow that helps review whether audit evidence satisfies control requirements.

## Business Problem

Security and compliance teams must provide complete, accurate, and timely evidence for audits. Missing or unclear evidence can delay audit response and increase compliance risk.

## Agent Use Case

The AI agent reviews fictional audit requests and generates evidence checklists, control owner questions, and validation steps for audit readiness.

## Skills Demonstrated

- Audit evidence review
- Control validation
- Access review support
- Security QA
- Documentation quality review
- Compliance readiness
